FTP Server command fuzzing

This script can be used to do fuzzing against an FTP server.
It supports the following commands:

  • ABOR
  • ACCT
  • ALLO
  • APPE
  • CWD
  • DELE
  • DIR
  • FORM
  • GET
  • HELP
  • LIST
  • MACDEF
  • MDELETE
  • MDIR
  • MGET
  • MKD
  • MLS
  • MODE
  • MODETIME
  • MPUT
  • NEWER
  • NLST
  • NMAP
  • MTDM
  • NTRANS
  • PUT
  • RECV
  • REGET
  • REMOTEHELP
  • REMOTESTATUS
  • REST
  • RESTART
  • RETR
  • RMD
  • RNFR
  • RNTO
  • QUOTE
  • SEND
  • SITE
  • SIZE
  • STAT
  • STOR
  • STRU
  • UMASK
